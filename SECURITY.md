# Security policy

## Supported version

Security fixes are applied to the latest version on the default branch.

## Reporting a vulnerability

Please use GitHub’s private vulnerability reporting for this repository. Do not include private journal text, passwords, or other sensitive personal information. We will acknowledge a valid report when possible and coordinate disclosure after a fix is available.

Cura has no backend and stores journal text locally, which limits its attack surface but does not make browser storage appropriate for secrets or highly sensitive records.
