# Accessibility

Cura targets WCAG 2.2 Level AA. The current release includes semantic landmarks and headings, keyboard-operable controls, visible focus, a skip link, reduced-motion support, bilingual document language updates, and responsive reading layouts.

Accessibility is an ongoing practice. If something prevents you from reading or writing, please open a GitHub issue with the page, browser or assistive technology, and the barrier you encountered. Personal or medical information is not required.
